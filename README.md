# Aether

> Personalized Environmental Monitoring, Worn Every Day.

Aether is an open-source wearable air quality monitoring platform built around the ESP32-S3, Flutter, and Firebase.

## Features

- PM2.5 Monitoring
- Temperature & Humidity
- Bluetooth Low Energy
- Mobile App
- Historical Analytics
- Cloud Synchronization
- Smart Alerts
